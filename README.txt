Este es el perfil de instalación en Español.
Seguir los pasos siguientes:

copiar el contenido de spanish.txt en un archivo spanish.make

subir el archivo tu servidor luego ejecutar

para descargar los archivos en una carpeta:
[~/public_html/]# drush make spanish.make carpeta

para descargar los archivos en el lugar donde se encuentra:
[~/public_html/]# drush make spanish.make .


esperar la descarga y listo.
